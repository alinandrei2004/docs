Retrieve new lists of Unikraft components, libraries and packages.

```
kraft pkg update [FLAGS]
```

### Examples

```
$ kraft pkg update

```

## Options

```
  -h, --help             help for update
  -m, --manager string   Force the handler type (default "manifest")
```

## Options inherited from parent commands

```
  -K, --kraftfile string   Set an alternative path of the Kraftfile
```
