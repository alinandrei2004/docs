---
title: "kraft cloud instance delete"
description: Delete an instance

Delete a KraftCloud instance.

```console
kraft cloud instance delete UUID
```

### Examples

```console
# Delete a KraftCloud instance
$ kraft cloud instance delete fd1684ea-7970-4994-92d6-61dcc7905f2b

```

## Options

```text
      --all             Stop all instances
  -h, --help            help for delete
  -o, --output string   Set output format (default "table")
```
